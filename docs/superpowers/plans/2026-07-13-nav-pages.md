# New Nav Pages Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Services, Results, Insights (+ article pages), and FAQ pages, and surface all of them in the top nav.

**Architecture:** Static Server Components in the App Router. Content is typed TS data — shared data (`insights`) in `src/lib/`, single-page data inlined in the page. All pages reuse the existing `globals.css` class system and the shadcn `<Button>`. `/services` becomes a real page and every `/#services` link (except the homepage's own section) is rewired to it.

**Tech Stack:** Next.js 16.2.10 (App Router, `src/`), React 19, TypeScript, Tailwind v4 + hand-written `globals.css`, lucide-react icons, shadcn `<Button>`.

## Global Constraints

- Next.js **16.2.10**. Dynamic pages use `params: Promise<{ slug: string }>` + `await params`; copy the exact typing style of `src/app/services/[slug]/page.tsx` (not the `PageProps` helper).
- All buttons render through `@/components/ui/button` `<Button>`, variants `primary`/`outline` only. Warm-monochrome; never pine/brass or any other style.
- Reuse existing `globals.css` classes. Add new CSS only where none fits (this plan adds exactly one class: `.card__meta`).
- Images use plain `<img>` with `{/* eslint-disable-next-line @next/next/no-img-element */}`, never `next/image`.
- Placeholder content must be meaningful and on-brand (tax/accounting/legal), never lorem ipsum, and any placeholder-heavy page carries a `<p className="placeholder-note reveal">…</p>`.
- **No unit-test framework exists** (static content site — adding one violates YAGNI). Per-task verification is: `npm run build` (type errors + route/static-param generation), `npm run lint`, and a dev-server render check at the route.
- Phone `98254 42028` → `tel:+919825442028`. Book-a-Consultation → `/contact`.

export function generateStaticParams() {

export async function generateMetadata(
  { params }: { params: Promise<{ slug: string }> },
): Promise<Metadata> {
  const { slug } = await params;
  const post = getPost(slug);
  if (!post) return {};
  return {
    title: `${post.title} · Patel Accounting & Legal Services`,
    description: post.excerpt,
  };
}
